# `users`

| Поле       | Тип         | Описание                                      |
| ---------- | ----------- | --------------------------------------------- |
| id         | SERIAL PK   | Уникальный идентификатор                      |
| nickname   | TEXT        | Ник пользователя                              |
| email      | TEXT UNIQUE | Email (уникальный)                            |
| password_hash | TEXT     | Hash пароля, если пользователь настроил пароль |
| birthday   | DATE        | День рождения                                 |
| is_verified | BOOLEAN   | Подтверждён ли email                          |
| timezone   | TEXT        | Таймзона пользователя                         |
| notifications_enabled | BOOLEAN | Включены ли email-напоминания        |
| reminder_days | TEXT     | Дни напоминаний через запятую: `0,1,3,7`      |
| reminder_hour | INTEGER | Локальный час отправки напоминаний            |


# `people`

| Поле    | Тип       | Описание                        |
| ------- | --------- | ------------------------------- |
| id      | SERIAL PK | Уникальный идентификатор        |
| user_id | INT FK    | Владелец (ссылка на `users.id`) |
| name    | TEXT      | Имя человека                    |
| date    | DATE      | Дата (например, день рождения)  |
| birth_month | INT   | Месяц дня рождения              |
| birth_day   | INT   | День месяца                     |
| birth_year  | INT   | Год рождения, если известен     |
| note    | TEXT      | Заметка                         |

# `shared_calendars`

| Поле          | Тип       | Описание                            |
| ------------- | --------- | ----------------------------------- |
| id            | SERIAL PK | Уникальный идентификатор            |
| owner_user_id | INT FK    | Создатель общего календаря          |
| slug          | TEXT      | Постоянная ссылка, например `pokrov` |
| name          | TEXT      | Название общего календаря           |
| created_at    | TIMESTAMP | Дата создания                       |


# `shared_calendar_members`

| Поле                  | Тип       | Описание                                  |
| --------------------- | --------- | ----------------------------------------- |
| id                    | SERIAL PK | Уникальный идентификатор                  |
| calendar_id           | INT FK    | Общий календарь                           |
| user_id               | INT FK    | Пользователь-участник                     |
| timezone              | TEXT      | Таймзона для личных напоминаний           |
| notifications_enabled | BOOLEAN   | Включены ли напоминания общего календаря  |
| reminder_days         | TEXT      | Дни напоминаний через запятую: `0,1,7`    |
| reminder_hour         | INTEGER   | Локальный час отправки                    |
| joined_at             | TIMESTAMP | Дата вступления                           |


# `shared_calendar_birthdays`

| Поле        | Тип       | Описание                                  |
| ----------- | --------- | ----------------------------------------- |
| id          | SERIAL PK | Уникальный идентификатор                  |
| calendar_id | INT FK    | Общий календарь                           |
| user_id     | INT FK    | Владелец записи; `NULL` для записей, добавленных админом вручную |
| name        | TEXT      | Имя участника                             |
| date        | DATE      | Совместимость со старой моделью даты      |
| birth_month | INT       | Месяц дня рождения                        |
| birth_day   | INT       | День месяца                               |
| birth_year  | INT       | Год рождения, если известен               |
| note        | TEXT      | Заметка                                   |
| edited_by_user_id | INT FK | Пользователь, который последним редактировал запись |
| created_at  | TIMESTAMP | Дата создания                             |
| updated_at  | TIMESTAMP | Дата обновления                           |
